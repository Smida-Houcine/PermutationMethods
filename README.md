# PermutationMethods
A collection of EViews programming methods for permutation generation.  
Repository for the EViews Add-in: **PermutationMethods**.  
**Description:**  
**PermutationMethods** is an EViews Add-in providing a collection of algorithms for permutation generation. The Add-in is packaged as a **.aipz** installer, enabling one-click integration into the EViews environment. You can also download it from: http://www.eviews.com/Addins/permutationmethods.aipz  
**Installation:**  
To install the Add-in, simply run the **.aipz** installer and follow the on-screen instructions within EViews.  
**Usage:**  
Once installed, the Add-in **appears** in the EViews Add-ins menu as “**Permutation Generation Methods**”.  
**Requirements:**  
• EViews version: Implemented in EViews 13 and tested in EViews 10.  
• Operating System: Windows 10/11.
